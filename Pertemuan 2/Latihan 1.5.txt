anda, pernahkah terjadi error ketika ada kesalahan
kode?
2. Cobalah ubah nilai meta refresh <meta HTTP-EQUIV="REFRESH"

Jawan 1.5
1. Tidak terjadi error apabila kode tidak sesuai
2. Setelah nilai meta refresh diubah seperti disoal, maka yang terjadi itu
halaman padaa saat terefresh akan langsung diarahkan ke http://google.co.id .
kegunaan meta data dokumen adalah untuk memberikan informasi mengenai
halaman web.
3. fungsi thead : untuk inisialisasi kolom, tbody : untuk inisialisasi
baris, td : untuk menandakan suatu cell table untuk diisi, th : setelah
diinisialisasi dengan thead maka untuk mengisi cell kolom dapat menggunakan th,
tr: untuk mendefinisikan baris pada table
